Command Name <--------------------------------------------------------> Comment
----------------------------------------------------------------------------------------------------------------------------------------------------------------
1.  git clone "link of the repository" <------------------------------> To clone the repository to the local PC from the Cloud Server 
2.  git status <------------------------------------------------------> To know the current status of your branch. eg. how many new files are added or changed
3.  git add . <-------------------------------------------------------> To add all the new changes to the git repository
4.  git commit -s -m "your message" <---------------------------------> To commit your new changes to the repository with message "your message", here, '-s' means commit will be signed off with your name and email and '-m' means message
5.  git commit --amend <----------------------------------------------> To add new changes to the last commit
6.  git pull  <-------------------------------------------------------> To get the latest changes in the Cloud Server
7.  git pull --rebase <-----------------------------------------------> To keep your commit to the top of the repository
8.  git push origin HEAD <--------------------------------------------> To send your commit to the Cloud Server
9.  git config --global user.name "Your Name" <-----------------------> To set your name globally
10. git config --global user.email "Your email" <---------------------> To set your email globally
11. git log <---------------------------------------------------------> To see all the log message of the repository
12. git reset --hard "commit-id" <------------------------------------> To reset the repository to the desired commit-id
13. git remote set-url origin "url" <---------------------------------> To change the connection type from ssh to https or https to ssh
14. ssh-keygen <------------------------------------------------------> To generate the key for ssh connection
15. vim ~/.ssh/id_rsa.pub <-------------------------------------------> To see the public key generated by command 13
16. notepad ~/.ssh/id_rsa.pub <---------------------------------------> To see the public key generated by command 13
